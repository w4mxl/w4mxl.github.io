ButterKnife 的使用

发表于 2015-05-23 | 更新于 2016-01-14 | 分类于 Android | 评论数： | 阅读次数：


ButterKnife 是一个通过对 Android View 添加注解的方式来提高开发效率的开源库。

1. 配置

在项目中配置 ButterKnife 有下面几种方式:

JAR

旧式导 jar 包:Butter Knife v6.1.0 JAR

MAVEN

如果你使用 maven，可以添加 butterknife 库作为依赖

2.3 ViewHolder INJECTION

也可以简化 list adapter 中的 View Holder 模式。

2.4 LISTENER INJECTION

可以对事件监听进行注入 (方法的参数可以按需添加)

可以一次指定多个 id，同时加事件

可以给 listview 的 item 点击事件添加注入

自定义的 view 不用指定具体 id 也可以绑定它的 listeners

2.5 OPTIONAL INJECTIONS

通常情况下不会出现 the target view cannot be found 情况，但为了避免出现而报异常，可以在变量或者方法前面加上 @Optional 注解

2.6 其他

有一些 View, Activity, or Dialog 可能还是会去 find views id，butterknife 提供了 findById 方法，它会自动强转为正确类型。
