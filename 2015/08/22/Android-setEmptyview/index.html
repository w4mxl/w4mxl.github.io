1. 介绍

使用 listview、gridview 等列表形式的 view 时，如果后台 server 返回的数据为空，通常我们需要为页面增加 empty view （俗称空页面），使得页面显示起来更友好，这个时候就用到了 setEmptyView () 方法。


▲ setEmptyView () 是 AdapterView 中的一个方法，而我们常用到的 ExpandableListView, GridView, ListView 等都是继承之 AdapterView 的，可以直接调用 setEmptyView () 方法。
